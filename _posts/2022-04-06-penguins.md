The columns of interest for this tutorial are the following:
- Species

## ยง3. Manipulating Data

We are interested in the relationship between culmen length and culmen depth by species. However, the species names are pretty long and clunky. To solve this, we can shorten the species names.


```python
penguins['Species'] = penguins['Species'].str.split().apply(lambda x: x[0])
```

## ยง4. Plotting Data

We can use the Seaborn package to easily create a scatterplot. Note that we use the Species column for the `hue` argument to visually identify the species.


```python
sns.scatterplot(x = 'Culmen Length (mm)', y = 'Culmen Depth (mm)', data = penguins,
                hue = 'Species', s = 50, linewidth = 0, alpha = 0.7)
plt.title('Culmen Length vs. Depth by Species')
plt.show()
```
